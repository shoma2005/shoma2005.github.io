# LoopTrainer

動画学習・繰り返し視聴に特化した軽量動画プレーヤーです。
**Lite（無料）** / **Standard** / **Pro** の3エディションがあります。

## インストール

1. インストーラー（`LoopTrainerXxx_Setup.exe`）をダウンロード
2. インストーラーを実行（管理者権限は不要）
3. インストール時に「動画ファイルをLoopTrainerに関連付ける」をチェックすると、動画ファイルをダブルクリックで開けるようになります

## 使い方

### 動画を開く

以下のいずれかの方法で動画を開けます：

- **開くボタン**をクリックしてファイルを選択
- メインウィンドウに動画ファイルをドラッグ&ドロップ
- エクスプローラーで動画ファイルをダブルクリック（ファイル関連付け設定時）
- 動画ファイルを右クリック → **プログラムから開く** → **LoopTrainer**

対応形式: MP4, MKV, AVI, MOV, WMV, FLV

### 再生コントロール

- **Space**: 再生/一時停止
- **左キー/右キー**: 5秒巻き戻し/早送り
- **上キー/下キー**: 音量調整（0〜150%）
- **N/P**: 次の動画/前の動画
- **S/F**: 再生速度ダウン/アップ
- **R**: 画面を90度回転
- **L**: ループモード切り替え
- **F11**: 全画面表示の切り替え
- **A/B/C**: ABループのA地点設定/B地点設定/クリア

> **ヒント**: ボタンにマウスを重ねると対応するショートカットキーがツールチップで表示されます。

## ABループ機能

特定の区間を繰り返し再生する機能です。語学学習・楽器練習・ダンス練習などに便利です。

1. 繰り返したい区間の開始位置で **Aボタン**（または**Aキー**）を押す
2. 終了位置で **Bボタン**（または**Bキー**）を押す
3. A〜B区間が自動的にループ再生されます
4. シークバー上のマーカーをドラッグして微調整可能
5. **クリアボタン**（または**Cキー**）でABループを解除

**ループモードとの連携:**

| ループモード | B地点に達したとき |
|-------------|-----------------|
| ループ: オフ | 停止 |
| ループ: オン | A地点に戻って繰り返し |
| ループ: すべて | 次の動画に移動 |

## プレイリスト機能

**プレイリストボタン**をクリックしてプレイリストウィンドウを開きます。

- **ファイル追加**ボタン、またはドラッグ&ドロップで動画を追加
- 動画をダブルクリックで再生
- ドラッグ&ドロップで再生順序を変更
- **クリア**: 選択した動画を削除 / **全てクリア**: リスト全体をクリア

### サムネイル表示

**表示** → **サムネイル表示**で各動画のサムネイルを表示できます。

### プレイリストの保存・読み込み

**プレイリスト** → **プレイリスト保存**でよく使う動画リストを保存できます（Standard/Pro版: 無制限保存）。

## Pro版専用機能

### フレームコマ送り・コマ戻し

- **`,` キー**: 1フレーム戻し
- **`.` キー**: 1フレーム送り

一時停止中に1フレームずつ正確に移動できます。

### 外部字幕読み込み

**Pro機能** → **字幕ファイルを読み込む**から .srt / .ass / .vtt 形式の字幕ファイルを読み込めます。

### スクリーンショット

**Pro機能** → **スクリーンショット**（または **I キー**）で現在のフレームをPNG形式で保存します。
保存先: `ピクチャ\LoopTrainer\`

### ABループ保存・管理

1つの動画に最大20個のABループを名前をつけて保存できます。

- **Pro機能** → **ABループを保存**: 現在のABループを名前をつけて保存
- **Pro機能** → **ABループ一覧**: 保存済みABループの一覧・適用・削除

### クリップ切り出し

**Pro機能** → **クリップを書き出し**で、現在のABループ区間をMP4ファイルとして保存できます。
前回の書き出し先フォルダを記憶します。

### フォルダ一括登録・シャッフル・M3U (プレイリストウィンドウ)

- **プレイリスト** → **フォルダを追加**: フォルダ内の動画ファイルをまとめてプレイリストに追加
- **🔀 シャッフルボタン**: ランダム順で再生
- **プレイリスト** → **M3Uとして書き出し / M3Uを読み込む**: 他のプレイヤーとプレイリストを共有

### 視聴統計ダッシュボード

**Pro機能** → **視聴統計**で、動画ごとの再生回数・累計視聴時間・最終再生日時を確認できます。

### カスタム速度入力

**再生速度** → **カスタム速度入力**から 0.05x〜10.0x の範囲で任意の速度を直接入力できます。

## 設定の自動保存

**動画ごとの設定**（Standard / Pro版のみ）:
- 音量
- 再生速度
- ABループ位置（A地点・B地点）
- 画面回転角度

**アプリ全体の設定**（全エディション）:
- ウィンドウの位置とサイズ
- プレイリストウィンドウの位置とサイズ・表示状態
- ループモード
- 表示モード（パス/サムネイル）
- 言語設定

## トラブルシューティング

**音が出ない**
→ 上下キーで音量を調整、またはWindowsのボリュームミキサーを確認してください。

**動画が開けない**
→ 対応形式（MP4/MKV/AVI/MOV/WMV/FLV）と、ファイルが破損していないかを確認してください。

**ABループがうまく動作しない**
→ A地点 → B地点の順に設定し、クリアボタンで一度リセットしてから再設定してください。

**サムネイルが表示されない**
→ サムネイル生成には時間がかかる場合があります。動画ファイルが破損していないかも確認してください。

**全画面表示が解除できない**
→ F11キーを押すか、マウスを画面上部に移動して解除ボタンをクリックしてください。
